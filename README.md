

## **PROJECT NAME:**

**Skin Cancer Classification using EfficientNetB3**


---

## **PROJECT DESCRIPTION:**

This project focuses on classifying skin cancer images into 7 categories using the HAM10000 dataset. A deep learning model based on EfficientNetB3 is used with transfer learning to improve accuracy and efficiency. The system is also evaluated using performance metrics and explainable AI techniques like Grad-CAM.  

---

## **DATASET:**

HAM10000 dataset is used which includes dermatoscopic images of skin lesions. The dataset contains **7 classes** and is processed using **EfficientNetB3** for feature extraction and classification.

- HAM10000_metadata.csv: Contains image IDs and labels  
- HAM10000_images_part_1: First set of images  
- HAM10000_images_part_2: Second set of images
  and link of dataset /kaggle/input/datasets/kmader/skin-cancer-mnist-ham10000
